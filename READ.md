# Iris Power BI Dashboard

## Project Overview
This project showcases an interactive Power BI dashboard built using the Iris Flower dataset provided through the University of Nevada, Reno Data Analytics Program using the NCLab platform. The dashboard enables users to explore species characteristics through interactive visualizations and filters.

## Dashboard Features
- Interactive slicers
- KPI metrics
- Species comparison
- Scatter plots
- Bar charts
- Dynamic filtering

## Tools Used
- Power BI Desktop
- Microsoft Excel
- Data Visualization
- Data Modeling
